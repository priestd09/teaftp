# teaftp

Simple, read-only TFTP server.

### Requirements

    Go >= 1.11

### Installation

    go get github.com/xyproto/teaftp

### Running

In the directory where you wish to share files:

Either:

    sudo ./teaftp

Or as root or with the correct Linux capabilities:

    ./teaftp
